Frequently asked questions.

Use the calculator as a starting point. A Sighthound engineer can help you tune node counts, camera mixes, and software options, validate performance, and refine your total cost of ownership model for each site.

How accurate is the calculator?

The calculator reflects current list pricing for compute nodes, standard IP cameras, smart cameras, and the optional software add-on billed at $30/month per camera. Taxes, shipping, and optional support plans are not included and will be added to a formal quote.

How many cameras can a node support?

Each Sighthound compute node is sized to support up to 4 cameras in total, across both standard IP and smart camera types. The calculator auto-suggests nodes using this rule (ceil of cameras / 4) and highlights when your configuration exceeds recommended capacity.

What is the difference between Standard IP and Smart Cameras?

Standard IP Cameras are cost-effective devices for basic coverage and recording. Smart Cameras are AI-ready, premium devices with on-device analytics and are ideal when you need richer, real-time computer vision at the edge. Both count equally toward node capacity.

How is the software billed?

The software add-on is billed at $30/month per camera and is kept separate from your one-time hardware purchase. The calculator shows hardware totals under “Due today” and software under “Monthly” so you always see how ongoing software affects total cost of ownership.
